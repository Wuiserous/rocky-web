Common Fixes

Google sign-in keeps asking again

Make sure cookies and popups are allowed for getrockyapp.com. If you are using a private browser window, try a normal browser window.

Windows blocks the installer

Windows may warn about new apps. Confirm you downloaded Rocky from getrockyapp.com and follow the Windows prompt only if you trust the file.

The app does not open after install

Restart Rocky from the Start menu. If it still does not open, email support with your Windows version and Rocky version.

I want to report a bug or request a feature

Email support@getrockyapp.com with steps to reproduce, screenshots if useful, and what you expected Rocky to do.
